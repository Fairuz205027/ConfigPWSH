### Config Oh-My-Posh 

> 1. Download Oh-My-Posh

> 2. Rubah config dari Microsoft Powershell 

> 3. Gunakan themes oh my posh sesuai dengan path yang lu pakai

-------------------------------------------------------------------------------------------

### Inisialisasi custom themes
```
oh-my-posh init pwsh --config "C:\Users\<user>\themesps\japan.omp.json" | Invoke-Expression
```

Location dir themes oh my posh replace with new config
